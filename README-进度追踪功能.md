## 💡 核心特性

### 🎯 动态识别关键词

系统会实时识别流式输出中的关键词：

| 关键词 | 提示变化 |
|-------|---------|
| `"character_models"` | → "正在生成角色模型..." |
| `"scene_models"` | → "正在构建场景描述..." |
| `"cover"` | → "正在设计封面..." |
| `"page_number": 5` | → "正在生成第 5 页..." |

### 📊 具体进度显示

不再是模糊的"正在生成..."，而是：
- ✅ "正在生成第 5 页..."
- ✅ "正在生成插图 3/20"

### ⚡ 无固定等待

- ❌ 改进前：固定等待10秒
- ✅ 改进后：根据内容自动推进

## 🔧 快速集成

### 对于前端开发者

```javascript
// 1. 引入组件
const progress = new GenerationProgress();

// 2. 显示进度窗口
progress.show(bookId, 1);

// 3. 在接收流式数据时更新进度
reader.onData((chunk) => {
    progress.updateScriptProgress(chunk);
});

// 4. 图片生成时更新进度
for (let i = 1; i <= totalPages; i++) {
    await generateImage(i);
    progress.updateImageProgress(i, totalPages);
}

// 5. 完成
progress.complete(() => {
    console.log('完成！');
});
```
